A collection of common shell configurations that I use.

## Installation

1. Homebrew requirements
   ```
   xcode-select --install
   ```

2. Install [Homebrew](https://brew.sh/)

3. Clone dotfiles
   ```
   git clone https://github.com/m-martinez/dotfiles ~/.dotfiles
   ```

4. Install dotfiles
   ```
   ~/.dotfiles/install.sh
   ```
